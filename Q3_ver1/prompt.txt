步骤5：定义双分支神经网络架构
python
class DualBranchNet(nn.Module):
    架构要求：
    1. 共享特征提取层：
       - 2-3个全连接层
       - 每层包含：线性层 + 批归一化 + ReLU激活 + Dropout
       - 隐藏层维度：128-256个神经元
    2. 两个独立预测分支：
       - 评委评分分支：输出维度1
       - 粉丝投票分支：输出维度1
       - 每个分支：2个全连接层，32-64个神经元
    3. 前向传播流程：
       - 输入 → 共享层 → 共享特征 → 两个分支 → 两个输出
步骤6：定义加权多任务损失函数
python
class WeightedMultiTaskLoss:
    要求：
    1. 分别计算两个任务的MSE损失
    2. 可调权重参数：alpha（评委权重）、beta（粉丝权重）
    3. 总损失 = alpha × MSE_judge + beta × MSE_fan
    4. 初始权重：alpha=1.0, beta=1.0
步骤7：训练循环实现
python
def train_model():
    核心组件：
    1. 优化器：Adam，学习率0.001
    2. 学习率调度器：ReduceLROnPlateau（当验证损失停滞时降低学习率）
    3. 早停机制：当验证损失连续10个epoch不下降时停止训练
    4. 模型检查点：保存验证集上表现最好的模型
    
    训练流程：
    1. 每个epoch：
       - 训练模式：前向传播 → 计算损失 → 反向传播 → 参数更新
       - 验证模式：前向传播 → 计算验证损失 → 不更新参数
    2. 记录指标：
       - 训练损失、验证损失
       - 两个任务各自的损失
       - 学习率变化

def classify_feature_impacts():
    特征影响类型分类：
    1. 协同影响：对评委和粉丝都有正面/负面影响
    2. 对立影响：对评委正面但对粉丝负面，或相反

    4. 无影响：对两个目标都无显著影响

    1. 识别"双赢特征"：能同时提升评委和粉丝评价
    2. 识别"权衡特征"：需要在评委和粉丝间做权衡
    3. 为不同类型选手提供策略建议
