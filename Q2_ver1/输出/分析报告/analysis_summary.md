## 方法论

### 核心量化指标

1. **粉丝相关性系数 (ρ)**: Spearman秩相关系数，衡量粉丝投票与最终结果的关联强度
   - 取值范围: [-1, 1]
   - 解读: 越接近1，越偏向粉丝

2. **粉丝贡献权重 (W)**: 通过线性回归反推的粉丝投票实际贡献占比
   - 取值范围: [0, 1]
   - 解读: 数值越大，粉丝影响力越大

3. **纯粉丝偏差 (D)**: 基于Kendall tau距离，衡量最终结果与纯粉丝意愿的偏离程度
   - 取值范围: [0, 1]
   - 解读: 数值越小，越接近纯粉丝排名

4. **排名相对差异 (ΔRank_rel)**: 两种方法下选手排名的相对差异
   - 计算: |Rank_R - Rank_P| / n × 100%
   - 解读: <20%轻微差异，≥40%显著差异

5. **粉丝挽救效应系数 (γ)**: 粉丝投票对评委低分的挽救作用
   - 计算: (Rank_Jonly - Rank_method) / Rank_Jonly × 100%
   - 解读: 正值越高，粉丝挽救效应越强

### 建模方法

- **排名法**: Score_R = 评委排名 + 粉丝排名（越小越好）
- **百分比法**: Score_P = 评委百分比 + 粉丝百分比（越大越好）
- **评委选择规则**: 每周从末两名中淘汰评委排名更差的选手

## 推荐与建议

### 投票机制推荐

基于量化分析结果，**推荐使用排名法**，理由如下：

1. 排名法在粉丝偏向性的三个核心指标上表现更优
2. 排名法能更好地反映粉丝意愿，提高观众参与感
3. 排名法对争议选手的粉丝挽救效应更强

### 规则建议

- 评委选择淘汰规则对整体机制影响有限，可根据节目需求选择是否采用
- 如果希望增加节目悬念和评委权威性，可考虑使用该规则
- 如果希望最大化粉丝影响力，建议不使用该规则
